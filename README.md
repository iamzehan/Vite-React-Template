## Vite + React Template with [React-Testing-Library](https://testing-library.com/)

**This template contains**
1. Tailwind CSS
2. ReactJS with Typescript (includes [Router](https://reactrouter.com/))
3. Vite Tooling
4. Complete Test Setup for React
5. Comes with [Material UI](https://mui.com/material-ui/)

### 1. Clone this repository
```shell
git clone https://github.com/iamzehan/Vite-React-Template.git
```
*Important: Remove the origin*
